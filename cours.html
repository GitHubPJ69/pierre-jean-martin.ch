Mes cours

Quatre directions, un seul prof.

Choisis par sujet, on adapte le contenu à ton niveau et tes objectifs.

Python

Le langage le plus polyvalent, pour automatiser, analyser et créer.

  • Fondamentaux solides : variables, fonctions, structures de données, POO
  • Automatisation de tâches métier (Excel, fichiers, web, API)
  • Initiation à la data : pandas, numpy, visualisation
  • Préparation aux examens EPFL, HES, gymnase

Pour : étudiants, professionnels en reconversion, ingénieurs qui veulent monter en compétences.

C / C++

Les fondations de l'informatique, démystifiées.

  • Algorithmique et résolution de problèmes
  • Pointeurs, gestion mémoire, références (sans paniquer)
  • Programmation orientée objet en C++
  • Préparation ciblée examens EPFL, HEIG-VD, HEIA-Fr

Pour : étudiants en école d'ingénieur, gymnasiens en option informatique, devs qui veulent comprendre ce qui se passe sous le capot.

IA & LLM

Passer d'utilisateur de ChatGPT à créateur de solutions intelligentes.

  • Comprendre l'écosystème : Claude, ChatGPT, Codex, Gemini, Mistral
  • Prompt engineering : obtenir des résultats fiables et structurés
  • Coder avec l'IA : Claude Code, Cursor, GitHub Copilot
  • Cas d'usage métier : automatisation, chatbot, traitement de documents

Pour : débutants curieux, professionnels en reconversion, cadres et indépendants. Aucun prérequis technique.
